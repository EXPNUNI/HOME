# void enuSetComponentHighlightOption\(bool )

void enuSetComponentHighlightOption\(bool )

#### Parameters

bool

true, false를 입력합니다.

#### Return Value

Type : bool

true, false를 반환합니다.


#### Remarks

로직 use 객체에 대하여 선택시 선택정보를 표시하는 하이라이트 표시에 대한 설정을 수행하는 함수이다.
기본설정값은 하이라이트 표시 모드이다.

#### Examples

```cpp
	enuSetComponentHighlightOption(false);
```
