# void enuSetComponentSymbolTableOption\(bool)

void enuSetComponentSymbolTableOption\(bool)

#### Parameters

bool

true, false를 입력합니다.

#### Return Value

Type : bool

true, false를 반환합니다.

#### Remarks

본 함수는 마우스 더블클릭 수행시 use의 로직객체 심볼 테이블 팝업속성을 설정하는 함수이다.
기본 설정값은 심볼 테이블 보여주기 모드이다.

#### Examples

```cpp
	enuSetComponentSymbolTableOption(false);
```
